# places-web

Its a React SPA for my pet project. Deployed on Firebase.

# Technologies used:

- React.js
- Google maps api

# Features

- JWT authorization
- Google maps integration
- Static files upload

# Installation

You'll need env vars to start an api. Example can be found in `nodemon.example.json`. After settting up env vars, run:

- `npm install`
- `nodemon app.js`
